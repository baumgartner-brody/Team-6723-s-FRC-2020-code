# Team-6723-s-FRC-2020-code
I've gone through it all and cleaned it up to the point where it might actually be readable. A lot of the commands and subsystems are near identical, most just have the basic input of the speed that the motor should run, but some have timers and run off encoder measures. The variables are named straight forward enough to be somewhat self explanatory. 

The way you use this is by uncompressing either the .zip or .rar and opening the folder in Virtual Studio Code. Run it by right clicking build.gradle and selecting "Build Robot Code" when actually connected to the robot you can hit "Deploy Robot Code." 
The official documentation can be found here: https://docs.wpilib.org/en/stable/docs/software/wpilib-overview/deploying-robot-code.html
